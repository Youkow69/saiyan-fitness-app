# Saiyan Fitness App

Mobile-first fitness app rebuilt from scratch with:
- personalized onboarding and automatic TDEE/macros
- recommended training programs
- guided workout logging with rest timer
- nutrition tracking and recipe suggestions
- progress analytics and DBZ-inspired gamification

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

The repository includes a GitHub Pages workflow. Pushing to `main` deploys the app automatically.
